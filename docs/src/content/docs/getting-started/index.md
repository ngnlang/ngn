Welcome to ngn. This short walkthrough gets you from zero to a running program.

## What you will do

- Install the ngn CLI
- Create a `main.ngn` file
- Run a simple "Hello, ngn!" program

:::note
The installer is not publicly available yet. The steps below show the intended flow and will work once the installer ships.
:::

## Start here

1. Follow the install steps.
2. Build your first program.
3. Pick a next guide based on what you want to build.

Ready? Start with [Install](/getting-started/install/).
